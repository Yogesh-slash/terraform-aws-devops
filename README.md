# 🚀 Production Terraform AWS DevOps Infrastructure with Jenkins CI/CD

## 📌 Project Overview

This project demonstrates a Production-Style AWS DevOps Infrastructure using modern DevOps tools and Infrastructure as Code (IaC).

The complete infrastructure is provisioned using Terraform and automated through Jenkins CI/CD pipelines. Configuration management is handled using Ansible, making the deployment process fully automated.

---

## 🛠️ Tech Stack

- AWS
- Terraform
- Jenkins
- Ansible
- Docker
- Kubernetes (Upcoming)
- Git & GitHub
- Linux (Ubuntu)

---

## 🚀 Features

- Infrastructure as Code using Terraform
- Modular Terraform Architecture
- Remote State Management using S3
- State Locking using DynamoDB
- Jenkins CI/CD Pipeline
- Dynamic Jenkins EC2 Agents
- Automatic Terraform Apply
- Automatic Ansible Inventory Generation
- Ansible Configuration Management
- Nginx Deployment Automation
- Secure SSH Authentication using Jenkins Credentials
